# Welcher Ermittlertyp bist du? — Hidden Games

## 🔗 Live-Link (Test): **https://hiddenfrancis.github.io/ermittlertyp-quiz/**

Mehrsprachiges Persönlichkeits-Quiz (**DE · EN · IT · FR · ES**): neun Fragen, am Ende ein Ermittlertyp, eine passende Fallempfehlung und ein teilbares Sharepic für Instagram & Co.

Einfach den Link oben öffnen oder weitergeben — funktioniert auf Handy, Tablet und Laptop, ohne Anmeldung.

---

*Technisch:* eine einzelne Datei (`index.html`) plus die Bilder in `img/`, kein Build-Prozess, gehostet über GitHub Pages. Änderungen am Code sind nach dem Hochladen ca. 1 Minute später live.
